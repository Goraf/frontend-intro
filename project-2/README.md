# Project-2

Convert .psd template into a webpage.

## Goals

* Create webpage with a proper html structure and CSS styles that will look exactly like an example provided in .psd template.

  (you can find .psd file in 'template' subdirectory)

* Webpage have to look correctly only on desktops. Doesn't have to be responsive.

* You can use [Adobe.assets](https://assets.adobe.com) to look-up content of the .psd file.
